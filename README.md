# Engineering Portfolio

This repository contains the source code for my personal engineering portfolio,
hosted via GitHub Pages.

The site highlights selected electrical engineering projects with a focus on
embedded systems, instrumentation, hardware design, and data-driven measurement.

## Live Site
https://denyshlya.github.io/portfolio/

## Featured Project
**Ski Pressure Testing System**  
A distributed force measurement system designed to characterize ski camber and
pressure distribution using 50+ sensors, custom electronics, embedded firmware,
and a PC-based analysis pipeline.

## Technologies Used
- HTML / CSS (static site)
- GitHub Pages + GitHub Actions
- Embedded systems (Arduino / PIC24)
- Sensor instrumentation and calibration
- Python / MATLAB for data analysis

## Purpose
This portfolio is intended for:
- Engineering recruiters and hiring managers
- Demonstrating hands-on design and validation experience
- Supporting applications for embedded, test, and instrumentation roles
